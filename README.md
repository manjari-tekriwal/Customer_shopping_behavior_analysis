# Retail Customer Behavior & Shopping Trends Analysis using SQL, Python and Power BI

Overview:
This project is a complete end-to-end Data Analytics workflow built on a real-world retail customer behavior dataset. The objective was to simulate how data analysts work in companies — from understanding business problems to cleaning data, analyzing trends, building dashboards, and presenting insights to stakeholders. The project combines Python, SQL, and Power BI to transform raw retail data into meaningful business insights and interactive visualizations.

This portfolio project demonstrates practical skills in:
1. Data Cleaning & Transformation
2. Exploratory Data Analysis (EDA)
3. SQL Business Querying
4. Dashboard Development
5. Business Reporting & Presentation
6. Business Problem Statement

A retail company wants to better understand customer purchasing behavior, shopping trends, product performance, and discount impact to improve decision-making and increase revenue.

The analysis focuses on:
1. Customer spending patterns
2. Product performance
3. Discounts and revenue trends
4. Shipping preferences
5. Subscriber vs non-subscriber behavior
6. Customer reviews and satisfaction
7. Dataset

Tools & Technologies:
1. Python (Pandas)
2. SQL (MySQL)
3. Power BI
4. Jupyter Notebook
5. Gamma AI (for presentation deck)
6. GitHub (project publishing & documentation)
   
Project Workflow / Steps:
1. Data Collection & Understanding
- Imported raw retail customer behavior dataset
- Explored dataset structure and columns
- Identified missing values and inconsistencies

2. Data Cleaning using Python
- Removed duplicates
- Handled null values
- Standardized column formats
- Converted data types
- Prepared clean dataset for analysis
  
3. Exploratory Data Analysis (EDA), to identify
- Customer purchase behavior
- Top-performing products
- Revenue trends
- Discount impact
- Customer satisfaction patterns.

4. SQL Analysis, to solve business problems such as
- Top-selling product categories
- Average customer spending
- Revenue by shipping type
- Subscriber vs non-subscriber comparison
- Discount effectiveness analysis
  
5. Power BI Interactive Dashboard
- KPI Cards
- Revenue Trends
- Product Performance Visuals
- Customer Segmentation
- Discount Analysis
- Shipping Insights
  
6. Reporting & Presentation
- Created a professional project report
- Built presentation slides using Gamma AI
- Published project files and documentation on GitHub
- Dashboard Features

Key insights:
- Customers using discounts still contributed significantly to revenue
- Certain product categories consistently received higher ratings
- Subscribers showed higher average spending behavior
- Express shipping customers had higher average purchase values
- Discount strategies influenced customer purchase frequency
